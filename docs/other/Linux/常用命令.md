### Linux常用命令
1. cd 路径
2. dd 删除
3. yy 粘贴
4. mkdir 创建文件夹  mkdir ./路径/文件夹名称
5. touch 创建文件
6. rm -r 删除
7. rmdir 删除空目录
8. :sp 文件名    同时打开多个文件进行操作
9. ctrl + w 切换窗口
10. echo &? 表示运行成功或者失败
11. pwd 查看当前路径
12. ls 查看文件夹
13. ls -a 查看当前目录下的文件夹 可显示隐藏文件
14. ls -al 可查看读写操作可显示隐藏文件
15. adduser 增加用户
16. su user 切换用户
17. mv ./路径 ./路径  表示移动xxx到xxx
18. cp 复制
19. cat  查看文件内容
20. more 查看文件内容
21. Find  可查看文件夹
